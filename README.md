## YealinkSaveConf

## Supported versions

    Yealink T21-e2 Yealink T21p-e2

    ver: 52.80.14.3
    ver: 52.84.0.125
    ver: 52.81.15.2
    ver: 52.84.14.5
    ver: 52.84.14.7


## To use

```
python3 -m pip venv venv
source venv/Scripts/activate
pip install -r ./req.txt
chmod +x yealink.py
yealink
```
